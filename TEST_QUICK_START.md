# 🚀 빠른 테스트 가이드

## 즉시 사용 가능한 테스트 질문

### 👍 추천 질문 (관계 O)

```
모험 81 챕터 보상이 뭐야?
→ 포코코로 보상 정보 + 관계 그래프

뱀파이어 제프는 뭐야?
→ 캐릭터 정보 + 연관 캐릭터 (좀비 포코타, 유령 노이)

메달은 어디에 쓰나요?
→ 상점 구매 정보 + 화폐 흐름

상점에서 뭘 살 수 있어?
→ 아이템 정보 + 메달 소비
```

### ✅ 기본 질문 (정보 O)

```
석판은 뭐야?
클리어하면 무엇을 받나요?
출석 보상은?
세이브 스테이지는 뭐야?
이벤트 스테이지는 어떻게 플레이해?
게임센터는 뭐야?
빙고는 어떻게 해?
구매 확인 팝업은 언제 나와?
메시지함은 어떻게 사용해?
크리스마스 보물자루는 뭐야?
```
